# Copy Unlocker

Chrome / Edge / Arc Manifest V3 扩展，用于恢复网页文本选择、复制、剪切快捷键和右键菜单。

## 功能说明

针对通过前端手段限制复制行为的网页，恢复浏览器默认的选择与复制能力。

### 能处理的限制

- CSS 禁止选择：`user-select: none`
- 行内事件属性：`oncopy`、`onselectstart`、`oncontextmenu` 等
- 页面脚本监听复制、选择、右键菜单、拖拽事件并阻止默认行为
- 页面脚本拦截 `Ctrl/Cmd + A`、`Ctrl/Cmd + C`、`Ctrl/Cmd + X`

### 不能处理的情况

- 需要登录、付费或特殊权限才能访问的内容
- 正文以图片、Canvas、视频等形式呈现（非可选中文本）
- 内容位于 Shadow DOM 等隔离结构内、扩展无法触及的部分
- 服务端加密或 DRM 保护的内容

## 安装

### 1. 打开扩展管理页

| 浏览器 | 地址 |
|--------|------|
| Chrome | `chrome://extensions/` |
| Edge | `edge://extensions/` |
| Arc | 菜单 → Extensions → Manage Extensions |

### 2. 开启开发者模式

在扩展管理页右上角打开**开发者模式**开关。

### 3. 加载扩展

1. 点击**加载已解压的扩展程序**（Load unpacked）。
2. 选择本目录：`copy-unlocker-extension`（即包含 `manifest.json` 的文件夹）。

安装成功后，扩展列表中会出现 **Copy Unlocker**。

## 使用方法

### 基本流程

1. **安装扩展**（见上文，只需做一次）。
2. **打开或切换到**需要复制内容的网页。
3. **刷新该页面**（`Cmd + R` / `F5）。  
   扩展在页面加载最早阶段注入，**已打开的旧标签页必须刷新后才会生效**。
4. 正常使用选择与复制：
   - 鼠标拖选文字
   - `Cmd + C`（Mac）或 `Ctrl + C`（Windows）复制
   - `Cmd + A` / `Ctrl + A` 全选
   - `Cmd + X` / `Ctrl + X` 剪切
   - 右键菜单复制

### 页面已经打开时怎么办？

如果目标网页在安装扩展**之前**就已经打开：

1. 确认扩展已安装并处于启用状态。
2. 切回该标签页，**刷新页面**。
3. 刷新后再尝试拖选或复制。

不刷新的话，扩展不会介入该页面的加载过程，复制限制通常仍然存在。

### 卸载或临时关闭

- **临时关闭**：在扩展管理页关闭 Copy Unlocker 的开关，然后刷新目标网页。
- **卸载**：在扩展管理页点击「移除」，然后刷新目标网页。

## 常见问题

### 安装后仍然无法复制？

按顺序排查：

1. **是否刷新了页面？**  
   这是最常见的原因。安装或启用扩展后，必须刷新目标标签页。

2. **扩展是否已启用？**  
   在扩展管理页确认 Copy Unlocker 开关为开启状态。

3. **能否选中文字？**  
   - 若**完全无法拖选**：可能是正文并非文本（见下条），或页面使用了扩展无法覆盖的特殊结构。
   - 若**可以选中但复制失败**：尝试右键复制，或检查是否被其他扩展冲突拦截。

4. **正文是否为图片 / Canvas / 视频？**  
   扩展只能恢复 HTML 文本的选择与复制。若内容是图片里的文字、Canvas 绘制或视频字幕流，需要其他方式（如 OCR、开发者工具查看网络请求等）。

5. **是否在 iframe 内？**  
   扩展已配置 `all_frames: true`，一般 iframe 内也会生效。若仍无效，可能是跨域 iframe 或特殊沙箱限制。

6. **是否与其他扩展冲突？**  
   可尝试在无痕窗口（仅启用 Copy Unlocker）或暂时禁用其他扩展后重试。

### 为什么必须「加载已解压的扩展程序」？

本扩展未上架 Chrome Web Store，以源码形式存放在仓库中。开发者模式下加载本地目录即可使用，修改代码后可在扩展管理页点击「重新加载」生效。

### 扩展会对所有网站生效吗？

是的。当前版本匹配 `<all_urls>`，安装后对所有网站生效。若你只希望对特定域名启用，需要自行修改 `manifest.json` 中的 `matches` 规则。

### 使用这个扩展合法吗？

扩展仅恢复浏览器原生的选择与复制行为，不破解登录、付费墙或服务端权限。请遵守目标网站的使用条款与版权规定，合理、合规地使用。

### 修改代码后如何更新？

1. 在扩展管理页找到 Copy Unlocker，点击「重新加载」（Reload）。
2. 刷新需要使用该扩展的网页。

## 技术说明

- Manifest V3，无额外权限声明
- 通过 Content Script + 页面脚本注入，在 `document_start` 阶段介入
- 使用 CSS 覆盖、`MutationObserver` 持续监听、事件捕获拦截等方式对抗常见的防复制手段

更完整的实现细节见 [`content.js`](./content.js) 与 [`unlock.css`](./unlock.css)。
